# 📊 커뮤니티 콘텐츠 대시보드

> 게임 커뮤니티 콘텐츠 성과를 프로젝트별로, 국가별로 관리하는 대시보드

담당 프로젝트와 해외 빌드, 여러 프로젝트의 콘텐츠 성과를 한눈에 비교합니다.
게임 커뮤니티에서 콘텐츠를 기획하고 배포하는 사람을 위해 주요 데이터를 한눈에 볼 수 있도록 구성하였습니다.

---

## 주요 기능

### 메인 대시보드
- 전체 프로젝트 조회수 합산 및 평균 참여율
- 성과 TOP 프로젝트 자동 하이라이트
- 플랫폼별 총 조회수 차트

### 프로젝트별 관리
- 프로젝트 단위로 콘텐츠 분리 관리 (예: 국내 운영 / 북미 마케팅 지원)
- 월별 콘텐츠 캘린더 — 날짜 클릭 시 해당 콘텐츠 상세 확인
- 플랫폼별 조회수 / 유형별 참여율 차트
- 조회수 상위 콘텐츠 랭킹 테이블

### 플랫폼별 참여율 공식 분리 적용

| 플랫폼 | 참여율 공식 |
|--------|------------|
| X, Instagram, Discord | (좋아요 + 댓글 + 공유) ÷ 조회수 × 100 |
| YouTube | (좋아요 + 댓글) ÷ 조회수 × 100 |
| 공식홈페이지 | 댓글 ÷ 페이지뷰 × 100 |

> YouTube는 외부 공유 집계가 불안정하고, 공식홈페이지는 좋아요/공유 기능이 없는 경우가 많아 각각 다른 기준을 적용합니다.

## 사용 방법

### 로컬 실행

```bash
git clone https://github.com/YOUR_USERNAME/content-dashboard.git
cd content-dashboard
open index.html
```

별도 설치 없이 브라우저에서 바로 실행됩니다.

### 프로젝트 추가

사이드바 하단 **+ 프로젝트 추가** → 이름, 설명, 색상 설정

### 콘텐츠 등록

프로젝트 진입 후 우측 상단 **+ 콘텐츠 등록** → 날짜, 플랫폼, 유형, 성과 지표 입력

## 기술 스택

- **Frontend**: Vanilla HTML / CSS / JavaScript (의존성 없음)
- **Chart**: Chart.js 4.4.1
- **Font**: Syne (display) + DM Sans (body) + DM Mono (mono)
